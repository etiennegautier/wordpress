Installing Theme

To start the theme installation first locate "cube.zip" within the package you already downloaded from Themeforest.

Unpack that file to "Desktop" where you can find it easily.

Than go to "Appearance" - "Themes" - "Install Themes" - "Upload" and point the file-dialog at "nordic.zip"

Creating Home Page With Slider

To create a default home page, you need to create a new Page first by going to Pages - Add New and choose "Home (with Slider)" as a page template from the drop-down on the right.

Immediately there should appear new meta-boxes.

Slider :

Use this portion to create the slider on the home page, click "Add Slide" to add a new slide, and insert there as many slides as you need.

When a new row appears use :

Image : To upload the slide image

Slide URL : If you want to navigate user somewhere, type a URL there

Slide Caption : Insert slide captions there, if you need you can separate them on multi captions

What We Do : 

This portion allows you to create services, click "Add Service" to add a new service, and insert there as many services as you need.

When a new row appears use :

Font-awesome : Enter the Font Awesome icon class here, you can grab them from their web-site

Service Title : Enter the service title on this field

Service Description : Enter the service description

Service URL : if you want to navigate user somewhere, than type a URL there

Latest Portfolio : 

You can configure the row on the bottom of this page where latest projects appear.

Enabling and Disabling Retina

This is an option, which acts not same as other options.

When you activate or de-activate this option, and on that period when new posts are included than there are chances the system did not create a retina version of images, and this is because we wanted to save space when users decide to not user Retina Images.

So when you activate or deactivate it please grab Force Regenerate Thumbnails and start the process.

Some plugins which should work are not working

You might experience some plugins which you supposed to work, they are not working.

This is because Nordic is a dynamic site, and it reloads only dynamic parts, so on those cases you have to Disable Ajax Load from Theme Options.
